# Development Tools

[English](development-tools.md) | [Deutsch](development-tools.de.md)

Software development & programming.

| Name | Description |
|------|-------------|
| [Beyond Compare](https://www.scootersoftware.com/) | Best tool for comparing files and folders - supports text, binary, images, archives |
| [Bruno](https://www.usebruno.com/) | Open-source Postman alternative - mature, large community, Git-friendly `.bru` format, no cloud lock-in |
| [DebugView](https://learn.microsoft.com/en-us/sysinternals/downloads/debugview) | Sysinternals tool for displaying `OutputDebugString` output - modern successor to the classic [DbMon](https://learn.microsoft.com/en-us/windows/win32/debug/outputdebugstring) |
| [dnSpy](https://github.com/dnSpy/dnSpy) | .NET decompiler and debugger - inspect, edit, and debug compiled .NET assemblies (active fork: [dnSpyEx](https://github.com/dnSpyEx/dnSpy)) |
| [Ghidra](https://github.com/NationalSecurityAgency/ghidra) | Open-source reverse engineering framework released by the NSA - free alternative to IDA Pro |
| [ILSpy](https://github.com/icsharpcode/ILSpy) | Open-source .NET assembly browser and decompiler from the SharpDevelop team |
| [Jexus Manager](https://github.com/jexuswebserver/JexusManager) | Open-source IIS manager - finally a graphical UI for IIS Express |
| [MailHog](https://github.com/mailhog/mailhog) | Local SMTP test server with web UI - intercepts outgoing mail instead of delivering it |
| [Nuitka](https://nuitka.net/) | Python compiler - translates Python into native C and produces standalone executables that run without a Python installation |
| [NVDA](https://www.nvaccess.org/) | Free open-source screen reader - essential for accessibility testing |
| [nvm](https://github.com/nvm-sh/nvm) | Node Version Manager - install and switch between multiple Node.js versions per shell (Unix/macOS; for Windows see [nvm-windows](https://github.com/coreybutler/nvm-windows)) |
| [ReSharper](https://www.jetbrains.com/resharper/) | Must-have extension for Visual Studio - code analysis, refactoring, navigation |
| [Rider](https://www.jetbrains.com/rider/) | Cross-platform .NET IDE from JetBrains - the modern alternative to Visual Studio |
| [Ruff](https://docs.astral.sh/ruff/) | Extremely fast Python linter and formatter written in Rust - replaces Black, isort and flake8 in a single tool |
| [WinMerge](https://winmerge.org/) | Free open-source diff and merge tool - solid budget alternative to Beyond Compare |
| [Yaak](https://yaak.app/) | Postman alternative from the Insomnia founder - newer than Bruno, modern UI, plugin system, also local-first |
