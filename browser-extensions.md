# Browser-Extensions

Erweiterungen für Webbrowser.

| Name | Beschreibung |
|------|--------------|
| [Archive.ph Sender](https://github.com/michaelblaess/browser-extensions/tree/main/archive-extension) | Speichert Webseiten mit einem Klick bei archive.ph |
| [Dark Reader](https://darkreader.org/) | Dark Mode für alle Webseiten - schont die Augen |
| [Site Inspector](https://github.com/michaelblaess/browser-extensions/tree/main/site-inspector) | Zeigt Technologien, Cookies, Security-Headers und WHOIS einer Webseite |
