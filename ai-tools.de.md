Künstliche Intelligenz & Machine Learning Tools.

| [Ollama](https://ollama.com/) | LLMs lokal ausführen via CLI - schnell und einfach |
| [OpenClaw](https://github.com/openclaw/openclaw) | Experimentelles KI-Tool - risky aber spaßig |
