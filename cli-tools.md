# CLI Tools

[English](cli-tools.md) | [Deutsch](cli-tools.de.md)

Terminal applications, shell tools, and libraries for building TUIs (Text User Interfaces) - with a focus on the Textual ecosystem.

## Terminal & Shell

| Name | Description |
|------|-------------|
| [Oh My Posh](https://ohmyposh.dev/) | Cross-shell prompt with theme engine - originated in the PowerShell world, now cross-platform with a huge theme library |
| [Starship](https://starship.rs/) | Minimal, blazing-fast cross-shell prompt written in Rust - single binary, works in any shell |
| [Windows Terminal](https://github.com/microsoft/terminal) | Microsoft's modern terminal emulator - tabs, multiple shells, GPU-accelerated rendering, themable |

## Frameworks & Libraries

| Name | Description |
|------|-------------|
| [Rich](https://rich.readthedocs.io/en/stable/) | Python library for beautiful terminal output - colors, tables, markdown, syntax highlighting, progress bars |
| [Textual](https://textual.textualize.io/) | Python framework for terminal UIs - reactive, CSS-based styling, runs in the terminal and the browser |

## Textual Extensions & Widgets

| Name | Description |
|------|-------------|
| [textual-autocomplete](https://github.com/darrenburns/textual-autocomplete) | Dropdown autocomplete widget for Textual inputs |
| [textual-image](https://github.com/lnqs/textual-image) | Display images directly in the terminal - supports Sixel, Kitty and half-block rendering |
| [textual-slider](https://github.com/TomJGooding/textual-slider) | Slider widget for Textual - missing from the default set |

## Visual Effects

| Name | Description |
|------|-------------|
| [TerminalTextEffects](https://github.com/ChrisBuilds/terminaltexteffects) | Animation engine for terminal text - many ready-made effects (Matrix, Rain, Burn, etc.) |
| [textualeffects](https://github.com/ggozad/textualeffects) | Textual wrapper for TerminalTextEffects - brings the effects directly into Textual apps |
