# Windows Tools

[English](windows-tools.md) | [Deutsch](windows-tools.de.md)

Useful applications for Windows.

| Name | Description |
|------|-------------|
| [7-Zip](https://www.7-zip.org/) | Free archive manager - supports all common formats |
| [Docker Desktop](https://www.docker.com/products/docker-desktop/) | Container platform for Windows - essential for modern development |
| [Greenshot](https://getgreenshot.org/) | Lightweight screenshot tool with editor and direct upload |
| [KeePass](https://keepass.info/) | Open-source password manager with local database |
| [RustDesk](https://rustdesk.com/) | Free open-source alternative to TeamViewer for remote desktop |
| [TreeSize](https://www.jam-software.de/treesize) | Visualizes disk usage - quickly find space hogs on any drive |
| [VirtualBox](https://www.virtualbox.org/) | Free open-source virtualization platform - solid alternative to VMware Workstation |
| [VMware Workstation](https://www.vmware.com/products/workstation-pro.html) | Professional virtualization solution for Windows |
