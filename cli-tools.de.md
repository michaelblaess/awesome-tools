# CLI-Tools

[English](cli-tools.md) | [Deutsch](cli-tools.de.md)

Terminal-Anwendungen, Shell-Tools und Bibliotheken zum Bau von TUIs (Text User Interfaces) - mit Fokus auf das Textual-Ökosystem.

## Terminal & Shell

| Name | Beschreibung |
|------|--------------|
| [Oh My Posh](https://ohmyposh.dev/) | Cross-Shell Prompt mit Theme-Engine - kommt ursprünglich aus der PowerShell-Welt, mittlerweile plattformübergreifend mit riesiger Theme-Bibliothek |
| [Starship](https://starship.rs/) | Minimaler, blitzschneller Cross-Shell Prompt in Rust - eine Binary, läuft in jeder Shell |
| [Windows Terminal](https://github.com/microsoft/terminal) | Modernes Terminal von Microsoft - Tabs, mehrere Shells, GPU-Rendering, Themes |

## Frameworks & Bibliotheken

| Name | Beschreibung |
|------|--------------|
| [Rich](https://rich.readthedocs.io/en/stable/) | Python-Bibliothek für schöne Terminal-Ausgaben - Farben, Tabellen, Markdown, Syntax-Highlighting, Progress-Bars |
| [Textual](https://textual.textualize.io/) | Python-Framework für Terminal-UIs - reaktiv, CSS-basiertes Styling, läuft im Terminal und im Web |

## Textual-Erweiterungen & Widgets

| Name | Beschreibung |
|------|--------------|
| [textual-autocomplete](https://github.com/darrenburns/textual-autocomplete) | Dropdown-Autocomplete-Widget für Textual-Inputs |
| [textual-image](https://github.com/lnqs/textual-image) | Bilder direkt im Terminal anzeigen - unterstützt Sixel, Kitty und Half-Block-Rendering |
| [textual-slider](https://github.com/TomJGooding/textual-slider) | Slider-Widget für Textual - fehlt im Standard-Set |

## Visuelle Effekte

| Name | Beschreibung |
|------|--------------|
| [TerminalTextEffects](https://github.com/ChrisBuilds/terminaltexteffects) | Animations-Engine für Terminal-Text - viele fertige Effekte (Matrix, Rain, Burn, etc.) |
| [textualeffects](https://github.com/ggozad/textualeffects) | Textual-Wrapper für TerminalTextEffects - bringt die Effekte direkt in Textual-Apps |
